## Pre-requirements
- NodeJS v16.x.x
- React Native Environments https://reactnative.dev/docs/environment-setup
- Yarn (**recommended**) or NPM


## Setup

#### Javascript Dependencies

    yarn install

or

    npm i

#### iOS Dependencies

    cd ios
    pod install

#### Android Dependencies

dependencies will installed when building the binary

## Running


> iOS

    yarn ios

> Android

    yarn android
